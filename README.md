# Spring Batch Angular Monitor

A tutorial program on how to display the progress of a Spring batch job by use of a progress bar with Angular

## Getting Started

To run the program, 
 - download the repo, 
 - build with maven, 
 - go to the server target folder 
 - run with java -jar app.war
